# 개념 (Concepts)

훈민정음 해례본의 우주철학·언어학 개념.

---

## #concept:하사땅

```yaml
id: concept:하사땅
name: 하사땅
aliases: [하늘 사람 땅, 천지인, 三才]
english: Heaven-Human-Earth
type: 우주론적 삼재
status: confirmed
```

**정의:** 우주의 세 근원적 층위 — 하늘(天)·사람(人)·땅(地). 훈민정음 창제의 바탕 사상.

**대응:**
- 하늘 ↔ [[#letter:ㅇ]] (원) ↔ [[#letter:ㆍ]] (점) ↔ 밝음 ↔ 0/1
- 사람 ↔ [[#letter:ㅅ]] (세모) ↔ [[#letter:ㅣ]] (선) ↔ 이음 ↔ 1/3
- 땅   ↔ [[#letter:ㅁ]] (네모) ↔ [[#letter:ㅡ]] (선) ↔ 어둠 ↔ 2/2

**관련:** [[#concept:원세네]] · [[#concept:밝이어]] · [[#term:하사땅]]

## #concept:원세네

```yaml
id: concept:원세네
name: 원세네
aliases: [원 세모 네모, 원세모네모, ○△□]
english: Circle-Triangle-Square
type: 기하학적 삼원소
status: confirmed
```

**정의:** 세 가지 근원 도형 — 원(○)·세모(△)·네모(□). [[#concept:하사땅]]의 시각적 표현.

**대응:**
- 원 (○) → 하늘 → [[#letter:ㅇ]]
- 세모 (△) → 사람 → [[#letter:ㅅ]]
- 네모 (□) → 땅 → [[#letter:ㅁ]]

**관련:** [[#concept:점선면]]

## #concept:점선면

```yaml
id: concept:점선면
name: 점선면 (원소 기하학)
aliases: [점 선 원, 점 선 면, 점선원]
english: Point-Line-Plane (Elemental Geometry)
type: 기하학적 원소
status: confirmed
```

**정의:** 가운뎃소리의 세 바탕 — 점(·)·선(ㅣ ㅡ)·면. 우주를 이루는 기하학적 원소.

**대응:**
- 점 (·) → 하늘 → [[#letter:ㆍ]]
- 선 (ㅣ) → 사람 → [[#letter:ㅣ]]
- 선 (ㅡ) → 땅 → [[#letter:ㅡ]]

## #concept:다움

```yaml
id: concept:다움
name: 다움
aliases: [다섯 움직임, 五行, 오행]
english: Five Movements / Five Phases
type: 오행 사상
status: confirmed
```

**정의:** 다섯 움직임 — 물·불·나무·쇠·흙. 전통적 오행을 우리말 신조어로 표현.

**대응(소리):**
- 물(水) — 목구멍소리 ㅇㆆㅎ
- 불(火) — 혓소리 ㄴㄷㅌㄹ
- 나무(木) — 엄소리 ㄱㅋ
- 쇠(金) — 잇소리 ㅅㅈㅊ
- 흙(土) — 입술소리 ㅁㅂㅍ
