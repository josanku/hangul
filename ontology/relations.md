## 닿소리 파생 트리

```
ㅇ (Heaven/Circle) ──┐
                     ├─ ㆆ ─ ㅎ ─ ㆅ
                     └─ ㆁ (이체)

ㅅ (Human/Triangle) ─┐
                     ├─ ㅈ ─ ㅊ
                     └─ ㅿ (이체)

ㅁ (Earth/Square) ──┬─ ㅂ ─ ㅍ
                    │
                    ├─ ㄱ ─ ㅋ          (어금니 - 본래 별도 바탕)
                    │
                    └─ ㄴ ─ ㄷ ─ ㅌ ─ ㄹ  (혀 - 본래 별도 바탕)
```

**참고:** 정통 해례본 분류는 5계열(ㅇ·ㅅ·ㅁ·ㄴ·ㄱ)이지만, 한글정음에서는 [[#concept:원세네]] 3원소 우주철학에 따라 ㄱ·ㄴ 계열을 ㅁ(땅)에서 파생되는 것으로 본다.

## 홀소리 파생 트리

```
바탕자:
  ㆍ (Heaven, 점)
  ㅣ (Human, 수직선)
  ㅡ (Earth, 수평선)

ㅣ + ㆍ 결합 (사람·하늘):
  ㅏ = ㅣ에 · 오른쪽 (밝음)
  ㅓ = ·에 ㅣ 오른쪽 (어둠)
  ㅑ = ㅣ에 ·· 오른쪽 (밝음 거듭)
  ㅕ = ··에 ㅣ 오른쪽 (어둠 거듭)

ㅡ + ㆍ 결합 (땅·하늘):
  ㅗ = ㅡ에 · 위 (밝음)
  ㅜ = ·에 ㅡ 위 (어둠)
  ㅛ = ㅡ에 ·· 위 (밝음 거듭)
  ㅠ = ··에 ㅡ 위 (어둠 거듭)

총 11 가운뎃소리 = 바탕 3 + 사람결합 4 + 땅결합 4
```

## 핵심 명제 (Propositions)

P1. **정음 28자는 [[#concept:하사땅]] × [[#concept:원세네]]를 본떠 만들어졌다.**
   - 출처: [[#source:해례본-1446]], [[#section:03-changje-wonri]]

P2. **닿소리 바탕자는 [[#letter:ㅇ]] [[#letter:ㅅ]] [[#letter:ㅁ]]이며, 각각 원·세모·네모를 본뜬 것이다.**
   - 출처: [[#section:03-changje-wonri]]

P3. **가운뎃소리 바탕자는 [[#letter:ㆍ]] [[#letter:ㅣ]] [[#letter:ㅡ]]이며, 점·세움선·눕힘선의 [[#concept:점선면]] 기하학을 따른다.**

P4. **모든 가운뎃소리에 하늘(ㆍ)이 들어있는 것은, 하늘의 기운이 만물에 고르게 미친다는 우주관을 반영한다.**
