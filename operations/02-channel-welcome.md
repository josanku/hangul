## 2. 자동 응답 키워드 (5–7개 등록)

채널 관리자 → 자동 응답 → 키워드 추가.

| 입력 키워드 | 자동 응답 |
|-------------|-----------|
| `책` `book` `다운로드` `download` | 📖 한글정음 책 무료 받기 → wehome.me/trust/hangul |
| `영문` `english` `eng` | 🌐 HANGUL, The Cosmic Philosophy (English) → wehome.me/trust/hangul-en |
| `오픈채팅` `대화` `chat` `살롱` | 💬 한글정음 살롱 입장 → open.kakao.com/o/pvUHv1pi |
| `사이트` `웹` `홈` `site` | 🌐 한글정음 공식 사이트 → [도메인] |
| `우주` `철학` `ㅇㅅㅁ` `하사땅` | ○△□ 한글의 우주철학 안내 → [도메인]#triad |
| `메뉴` `?` `도움` `help` | 📋 메뉴: `책` 책 받기 · `영문` 영어판 · `오픈채팅` 살롱 · `우주` 철학 · `사이트` 홈 |
| `행사` `이벤트` `모임` | 📅 다가오는 행사 → [도메인]#newsFeed |
