## 4. 초대 작성·발송 가이드

| 단계 | 행동 |
|------|------|
| 1 | 명단 작성 — 가까운 지인(15명) + 관심 분야 인사(10명) + SNS 공개 |
| 2 | 메시지 개인화 — `${이름}` 부분은 손으로 직접 채우기 (성의) |
| 3 | 시간 분산 — 하루에 5명씩 (스팸으로 분류 방지) |
| 4 | 첫 주는 매일 새 사람 안내 + 기존 가입자 1:1 환영 |
| 5 | 가입한 사람에게 24시간 내 첫 메시지 (운영자 직접) |

### ❌ 피해야 할 패턴

- 단톡방에 일괄 초대 후 광고처럼 메시지 발송
- 같은 메시지 복사·붙여넣기로 100명 발송
- 답장 없는 사람에게 2번 이상 보내기

### ✅ 권장 패턴

- 한 명씩 이름 호출, 한 줄이라도 개인적 첨언
- “관심 없으시면 무시해 주세요” 한 줄 — 부담 덜기
- 가입 직후 채널 환영 메시지 + 오픈채팅 입장 안내
