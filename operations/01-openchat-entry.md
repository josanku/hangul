# 오픈채팅 입장 질문 · 환영 메시지 · 규칙

## 1. 입장 질문 설정 (스팸·광고 차단)

오픈채팅 관리자 화면 → 입장 질문 설정.

**질문 (택 1)**

| 옵션 A — 책 독자 확인 (추천) |
|---|
| ❓ 한글정음에서 ㅇ · ㅅ · ㅁ는 각각 무엇을 상징할까요? (한 단어씩) |
| ✅ 정답: `하늘 사람 땅` 또는 `하늘, 사람, 땅` 또는 `Heaven, Human, Earth` |

| 옵션 B — 가벼운 진입 |
|---|
| ❓ 한글정음에 어떻게 오시게 되었나요? (간단히 한 줄) |
| ✅ 정답: 자유 (스팸 자동 차단 효과는 약하지만 자기소개 효과) |

| 옵션 C — 짧은 다짐 |
|---|
| ❓ 한글의 우주철학에 관심이 있으신가요? |
| ✅ 정답: `네` 또는 `예` 또는 `있어요` |

**추천: 옵션 A** — 책을 읽거나 사이트를 본 사람만 자연스럽게 통과합니다.

## 2. 입장 환영 메시지

입장 직후 자동으로 보낼 메시지 (개인 메시지로 따로 보내거나 공지로):

```

이곳은 한글과 우주철학을 함께 나누는 자리입니다.

처음이시라면 ↓
📖 책 무료 받기 → wehome.me/trust/hangul

— 운영자 조산구 드림
```

## 4. 입장 후 첫 3일 운영 팁

| 시점 | 행동 |
|------|------|
| 입장 즉시 | 개인 1:1 메시지로 환영 (운영자가 직접) |
| 24시간 내 | 신규 회원의 자기소개에 운영자가 첫 댓글 (분위기 형성) |
| 3일 후 | 활동 없는 신규 회원에게는 가볍게 안부 |

→ 초기 3일이 잔류 여부를 결정합니다. **이때 운영자의 적극성이 가장 중요.**
