## 통합 흐름

```
       검색·SNS                    검색·추천
          ↓                           ↓
       사이트 (hangul.kr)
       ├── 💛 채널 친구추가  ←─ 가벼운 구독자 (다수)
       └── 💬 오픈채팅 입장  ←─ 적극 참여자 (5~10%)

       채널 환영 메시지 ─→ 오픈채팅 안내
       오픈채팅 공지   ─→ 채널·사이트 안내
```
